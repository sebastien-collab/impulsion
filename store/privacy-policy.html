Impulsion — Privacy Policy

Last updated: February 22, 2026

Overview

Impulsion is a browser extension that helps digital marketers and media buyers debug tracking pixels and tags on web pages. We are committed to protecting your privacy.

Data Collection

Impulsion does not collect, store, transmit, or share any personal data.

All data processing happens locally in your browser. Specifically:

  • Pixel detection data is stored temporarily in Chrome's session storage and is automatically cleared when you close the tab.
  • No data is ever sent to any external server.
  • No analytics or telemetry is collected about your usage.
  • No cookies are set by the extension.
  • No user accounts or authentication are required.

Permissions Explained

Impulsion requests the following browser permissions, each for a specific technical purpose:

  • webRequest — Monitor network requests to detect tracking pixel HTTP calls (e.g., requests to google-analytics.com, facebook.com/tr, etc.). No requests are modified or blocked.
  • host_permissions (all URLs) — Required to detect tracking pixels on any website you visit. The extension only reads request URLs to identify known pixel patterns.
  • scripting — Inject a content script into pages to intercept JavaScript tracking function calls (e.g., gtag(), fbq(), ttq.track()).
  • storage — Store detected pixel data in Chrome's session storage for the current browsing session only.
  • activeTab — Access the current tab's URL to associate detected pixels with the correct tab.

Third-Party Services

Impulsion does not integrate with any third-party services, APIs, or analytics platforms.

Data Retention

All data is stored in Chrome's session storage and is automatically deleted when:

  • You close the tab
  • You navigate to a different page
  • You close the browser
