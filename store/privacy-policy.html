impulsion.com — Privacy Policy

Last updated: February 22, 2026

Overview

impulsion.com is a browser extension that helps digital marketers and media buyers debug tracking pixels, detect tech stacks, and inspect web pages. We are committed to protecting your privacy.

Data Collection

impulsion.com does not collect, store, transmit, or share any personal data.

All data processing happens locally in your browser. Specifically:

  • Pixel detection data is stored temporarily in Chrome's session storage and is automatically cleared when you close the tab.
  • User preferences (language, theme, color history) are stored locally via Chrome's local storage and never leave your browser.
  • No data is ever sent to any external server.
  • No analytics or telemetry is collected about your usage.
  • No cookies are set by the extension.
  • No user accounts or authentication are required.

Permissions Explained

impulsion.com requests the following browser permissions, each for a specific technical purpose:

  • webRequest — Monitor network requests to detect tracking pixel HTTP calls (e.g., requests to google-analytics.com, facebook.com/tr, etc.). No requests are modified or blocked.
  • host_permissions (all URLs) — Required to detect tracking pixels and technologies on any website you visit. The extension only reads request URLs and DOM elements to identify known patterns.
  • scripting — Inject content scripts to intercept JavaScript tracking calls and provide user-initiated tools (Eye Dropper color picker, Font Inspector).
  • storage — Store detected pixel data in session storage (cleared per tab) and user preferences in local storage (language, theme).
  • activeTab — Access the current tab's URL to associate detected data with the correct tab and scope tools to the current site.
  • cookies — Read and clear cookies for the current domain only, when the user explicitly triggers the "Clear Cookies" action.
  • browsingData — Clear cached data for the current site only, when the user explicitly triggers the "Clear Cache" action.

Third-Party Services

impulsion.com does not integrate with any third-party services, APIs, or analytics platforms. The Ad Libraries tab provides links to external ad transparency tools (Meta Ad Library, Google Ads Transparency Center, etc.) but no data is shared with these services.

Data Retention

Detection data (pixels, events, consent, tech stack) is stored in Chrome's session storage and is automatically deleted when:

  • You close the tab
  • You navigate to a different page
  • You close the browser

User preferences (language, theme, color history) persist in local storage until the extension is uninstalled.
