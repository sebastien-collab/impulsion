## Full Description

### impulsion.com — Universal Pixel Debugger & Tech Stack Detector

impulsion.com instantly detects every tracking pixel and technology running on any webpage. Built by media buyers, for media buyers.

**Supported tracking platforms:**
- Google Tag Manager (GTM)
- Google Analytics 4 (GA4)
- Google Ads (conversion tags)
- Meta Pixel (Facebook)
- TikTok Pixel
- LinkedIn Insight Tag
- Pinterest Tag
- Snapchat Pixel
- Twitter/X Pixel
- Microsoft Advertising (Bing UET)

**5 tabs, one tool:**

**Pixels** — Real-time detection via 3 methods: JavaScript interception, DOM scanning, and network monitoring. Pixel ID extraction, event timeline with full arguments, per-platform color coding.

**Consent** — Google Consent Mode V2 checker. Verify default/update commands, detect missing V2 parameters (ad_user_data, ad_personalization), identify Basic vs Advanced mode, GTM internal consent state detection.

**Ad Libraries** — One-click links to Meta Ad Library, Google Ads Transparency Center, LinkedIn Ad Library, and TikTok Creative Center for the current page's domain. Instantly research any advertiser.

**Stack** — Simplified Wappalyzer-like tech stack detection. Identifies 30+ technologies: CMS (WordPress, Shopify, Wix, Webflow...), JS frameworks (React, Next.js, Vue, Angular...), CSS frameworks (Tailwind, Bootstrap...), CDN (Cloudflare, Vercel...), and tools (HubSpot, Hotjar, Stripe, Crisp...).

**Tools** — Eye Dropper color picker (get any color from the page with hex/RGB/HSL values + history), Font Inspector (hover to see font family, size, weight, line-height, color), Clear cookies & cache for the current site.

**Key features:**
- Bilingual interface (English / French)
- Clean dark UI — 400x600px popup
- Zero performance impact — lightweight, no external requests
- Everything processed locally in your browser

**Privacy:**
impulsion.com processes everything locally. No data is collected, stored, or sent anywhere. Fully open source on GitHub.

**Use cases:**
- Audit client websites for correct pixel implementation
- Verify tracking is firing on landing pages
- Debug conversion tracking issues
- Check Google Consent Mode V2 compliance
- Research competitors via Ad Libraries
- Identify what technologies a website uses
- Pick colors and inspect fonts on any page
