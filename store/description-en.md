## Full Description

### Impulsion — Universal Pixel & Tag Debugger

Impulsion instantly detects and displays every tracking pixel running on any webpage. Built by media buyers, for media buyers.

**Supported platforms:**
• Google Tag Manager (GTM)
• Google Analytics 4 (GA4)
• Google Ads (conversion tags)
• Meta Pixel (Facebook)
• TikTok Pixel
• LinkedIn Insight Tag
• Pinterest Tag
• Snapchat Pixel
• Twitter/X Pixel
• Microsoft Advertising (Bing UET)

**Features:**
• Real-time detection via 3 methods: JavaScript interception, DOM scanning, and network monitoring
• Pixel ID extraction — see the exact IDs configured on the page
• Event timeline — every tracking call logged with full arguments
• Google Consent Mode V2 checker — verify default/update commands, check for missing V2 parameters (ad_user_data, ad_personalization), detect Basic vs Advanced mode
• Clean dark UI with per-platform color coding
• Zero performance impact — lightweight, no external requests

**How it works:**
Click the Impulsion icon in your toolbar to see all tracking pixels detected on the current page. Expand each pixel card to view the captured events and their parameters.

**Privacy:**
Impulsion processes everything locally in your browser. No data is collected, stored, or sent anywhere. Fully open source on GitHub.

**Use cases:**
• Audit client websites for correct pixel implementation
• Verify tracking is firing on landing pages
• Debug conversion tracking issues
• Check Google Consent Mode V2 compliance
• Compare pixel setup across competitor sites
